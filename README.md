# AI Campus Assistant API (CampusGPT)

An AI-powered backend API for the AI Campus Assistant mobile application.
This system helps students, faculty, and visitors get instant answers to university-related queries using a Retrieval-Augmented Generation (RAG) architecture.

The API is built with FastAPI and integrates with AI models and a vector database to provide accurate, context-aware responses based on official university documents.

## Project Overview

The AI Campus Assistant provides:

Instant answers to campus-related questions
Retrieval of relevant university documents
AI-generated responses using LLMs
Feedback system for improving responses
Admin support for updating knowledge base

This project is part of a Final Year Project (FYP).
